200

HTTP Status Code

Means: Success / OK

Common status codes:
| Code | Meaning      |
| ---- | ------------ |
| 200  | OK           |
| 404  | Not Found    |
| 500  | Server Error |

Tells the browser what type of data it is receiving.
| Content-Type       | Meaning     |
| ------------------ | ----------- |
| `text/plain`       | Normal text |
| `text/html`        | HTML page   |
| `application/json` | JSON data   |

🧑‍💼 HOW INTERVIEWER ASKS (CODING QUESTION)
Interview Question:
Write a simple Node.js program to create a server that sends a response to the browser.
OR
Create a basic HTTP server in Node.js.
------------------------------------------------------------------------------------------------------------
// built in packages...node js is providing built in packages

//http module

//import the built in http module

const http = require('https');

// create the server
const server = http.createServer((req, res) => {
    res.writeHead(200, { "Content-Type": "text/plain" })
    res.end("Hello hi guys")

});

//define the port number
const PORT = 5001;


//start the server
server.listen(PORT, () => {
    console.log(`server is runnning at the port ${PORT}`)
})
