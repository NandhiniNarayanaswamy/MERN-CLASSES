                                                üè∑Ô∏è Array
                                                ----------
In JavaScript, an array is a special type of object used to store multiple values in a single variable, in an ordered way.
You can think of it like a list where each item has a position (called an index).
üîë Key Point
----------
1Ô∏è‚É£ Stores multiple values
Instead of creating multiple variables:
üí° Example:
let fruit1 = "Apple";
let fruit2 = "Banana";
let fruit3 = "Orange";
You can store them in one array:
let fruits = ["Apple", "Banana", "Orange"];

2Ô∏è‚É£ Indexed starting from 0
The first element has index 0, the second 1, and so on
üí° Example:
console.log(fruits[0]); // "Apple"
console.log(fruits[1]); // "Banana"

3Ô∏è‚É£ Can store any data type (even mixed types)
üí° Example:
let mixed = [1, "Hello", true, { name: "Nandhini" }, [10, 20]];

4Ô∏è‚É£ Dynamic in size ‚Äî You can add or remove elements anytime.
üí° Example:
fruits.push("Grapes"); // Adds to end
fruits.pop(); // Removes last element

5Ô∏è‚É£ Array is actually an object
üí° Example:
console.log(typeof fruits); // "object"
console.log(Array.isArray(fruits)); // true

üìç Creating Array
-----------------
// 1. Using array literal (most common)
let arr1 = [10, 20, 30];

// 2. Using the Array constructor
let arr2 = new Array(10, 20, 30);

// 3. Empty array and adding elements later
let arr3 = [];
arr3[0] = "Apple";
arr3[1] = "Banana";

üßê How array can be an Objects ‚ùì
-------------------------------
üí≠ Arrays are Objects with Numbered Keys
Internally, an array is like an object with integer-like keys:
let fruits = ["Apple", "Banana", "Orange"];

// Internally similar to:
let fruitsObj = {
    0: "Apple",
    1: "Banana",
    2: "Orange",
    length: 3
};
The keys (0, 1, 2) are just string properties of the object.
length is an automatic property that updates when you add/remove elements.

                                                üè∑Ô∏èfor-of & for-in
                                                ------------------
‚úîÔ∏è for...in loop
******************
Purpose: Loop through keys (property names) of an object or array.

üìú Syntax:
for (let key in object) {
    // code using key
}

üîë Key Points:
----------------
‚ú®Iterates over enumerable property names (keys) in an object.
‚ú®Works for objects and arrays, but not recommended for arrays when you care about order or performance.
‚ú®The key is always a string, even for array indexes.
‚ú®Will also include inherited properties (from prototype) unless you check with hasOwnProperty.

üí° When to use for...in
For objects when you want property names.
Avoid it for arrays if you only want values (use for...of instead).

üí° Example
________________________________________________________________

let person = { name: "Nandhini", age: 22, city: "Bangalore" };

for (let key in person) {
    console.log(key, person[key]);
}

O/P
name Nandhini
age 22
city Bangalore
___________________________________________________________________

‚úîÔ∏è for..of loop
-----------------
Purpose: Loop through values of an iterable (Array, String, Map, Set, etc.).

üìú Syntax:
for (let value of iterable) {
    // code using value
}
üîë Key Points:
‚≠ê Iterates over values, not keys.
‚≠ê Works only on iterables (Array, String, Map, Set, etc.).
‚≠ê Doesn‚Äôt work directly on objects (because objects are not iterable by default).

üí° Example
_______________________________________________

let fruits = ["Apple", "Banana", "Orange"];

for (let fruit of fruits) {
    console.log(fruit);
}

Apple
Banana
Orange
________________________________________________

üí° When to use for...of
For arrays, strings, maps, sets, or any iterable when you need values directly.

                                                üè∑Ô∏è for...each loop
                                                -------------------
üßê What is forEach?
forEach is a built-in array method in JavaScript that lets you execute a function once for every element in an array, in order.

It‚Äôs like saying:

‚ÄúFor each element in this array, run this function.‚Äù

üìú Syntax:-
array.forEach(function(element, index, array) {
    // code to run for each element
});

Parameters:

element ‚Üí The current value in the array (required).
index ‚Üí The index of the current element (optional).
array ‚Üí The array being traversed (optional).

üí° Example
let fruits = ["Apple", "Banana", "Orange"];

fruits.forEach(function(fruit) {
    console.log(fruit);
});
Output
-------
Apple
Banana
Orange

üí° Example ‚Äî With Index

 let fruits = ["Apple", "Banana", "Orange"];
fruits.forEach(function(fruit, index) {
    console.log(index, fruit);
});

Output:
0 Apple
1 Banana
2 Orange

üìåImportant Points about forEach

‚ú® Works only on arrays (and array-like objects with forEach method).
‚ú® Always executes for each element ‚Äî cannot skip or break out (use for or for...of if you need to break).
‚ú® Does not return a new array ‚Äî it always returns undefined.
‚ú® Used when you want to perform an action, not transform data
________________________________________________________________________________

‚ùå Example of trying to break:
let nums = [1, 2, 3, 4];

nums.forEach(num => {
    if (num === 3) break; // ‚ùå SyntaxError ‚Äî can't use break here
    console.log(num);
});
